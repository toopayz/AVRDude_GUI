# AVRDude_GUI
GUI (Graphical User Interface) Aplikasi untuk meng-upload *.hex file ke mikrokontroller  
dengan memanfaatkan konsole AVRDude.  

OS   : Windows 10 64 bit  
IDE  : Free Pascal Lazarus v2.0.0  
Date : 2019-10-14  
FPC Version : 3.0.4  
SVN Version : 60307
